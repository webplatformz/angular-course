Angular @defer

Deferrable views (@defer blocks) help reduce the initial bundle size by lazy load any Angular standalone component.

@defer ( on <behavior trigger>; when <condition trigger>; prefetch on <behavior trigger>; prefetch when <condition trigger> ) {
  <app-large-component/>
} @placeholder ( minimum? <duration> ) {
  <p>Waits, until loading is triggered</p>
} @loading ( minimum? <duration>; after? <duration> ) {
  <app-loading-spinner/>
} @error {
  <p>Error! The large component could not be loaded.</p>
}

Features

  • Supports triggers (e.g., visibility, user interaction, timers).
  • Allows prefetching for better performance.
  • Provides sub-blocks for placeholder, loading, and error handling.
